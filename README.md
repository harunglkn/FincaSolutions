# Finca-Solutions LeadHub

Professionelle B2B SaaS-Plattform für freie Autohändler:
Fahrzeug-Leads, Verkäuferantworten, Ankaufspreise, Termine und Tagesberichte
zentral in einem Dashboard verwalten.

## Status

In Entwicklung — Grundstruktur wird aufgebaut.

## Technik-Stack

- **Next.js** (App Router, TypeScript)
- **Tailwind CSS** für Design
- **Supabase** (geplant) — Login, CRM, Datenbank
- **PWA** — installierbar wie eine native App

## Projektstruktur

```
Fincar-LeadHub/
├── leadhub/          # Next.js-Anwendung
└── README.md
```

## Entwicklung starten

```bash
cd leadhub
npm install
npm run dev
```

Öffne dann http://localhost:3000 im Browser.
